# Media Extractor API

این API برای استخراج ویدیوها، تصاویر و فایل‌های صوتی از هر صفحه وب طراحی شده است.

## ویژگی‌ها

- استخراج خودکار ویدیوها (MP4, AVI, MKV و...)
- استخراج فایل‌های صوتی (MP3, WAV, OGG و...)
- استخراج تصاویر (JPG, PNG, GIF و...)
- نمایش حجم فایل‌ها
- دسته‌بندی بر اساس نوع فایل
- حذف فایل‌های تکراری
